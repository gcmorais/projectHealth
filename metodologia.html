Metodologia da Pesquisa

Abordagem do Estudo

Esta pesquisa foi desenvolvida utilizando uma abordagem mista, combinando métodos quantitativos e qualitativos para garantir uma análise abrangente dos hábitos de atividade física na comunidade. O estudo foi estruturado em três etapas principais: coleta de dados, campanha de conscientização e avaliação de impacto.

Coleta de Dados

Os dados foram coletados através de questionários aplicados em pesquisa de rua, distribuídos para uma amostra de 200 participantes, com idade entre 18 e 65 anos. O período de coleta durou 3 meses, garantindo uma representatividade adequada da população estudada. A coleta foi realizada em diferentes locais da comunidade para obter uma visão mais completa dos hábitos de atividade física.

Instrumentos de Pesquisa

O questionário foi composto por perguntas sobre:

  • Frequência de prática de atividade física
  • Duração média das sessões de exercício
  • Tipos de atividades físicas praticadas
  • Percepções sobre benefícios à saúde
  • Hábitos de busca por informações sobre saúde
  • Barreiras encontradas para a prática de exercícios
  • Preferências por locais de prática

Campanha de Conscientização

Após a análise inicial dos dados, foi desenvolvida uma campanha de conscientização que incluiu:

  • Distribuição de materiais educativos (folhetos e cartazes)
  • Orientações personalizadas sobre atividade física
  • Divulgação de informações no site do projeto
  • Indicação de programas e recursos disponíveis para a comunidade

Avaliação e Feedback

Para avaliar o impacto das ações, foi realizada uma pesquisa de satisfação com os participantes, incluindo:

  • Avaliação da qualidade das informações recebidas
  • Mudanças nos hábitos de atividade física
  • Sugestões para melhorias futuras
  • Nível de satisfação com as ações realizadas

Análise dos Dados

Os dados coletados foram analisados utilizando métodos estatísticos descritivos e análise de conteúdo para os dados qualitativos, permitindo a identificação de padrões e tendências nos hábitos de atividade física da população estudada. Os resultados foram organizados por faixa etária para melhor compreensão do perfil da comunidade.
